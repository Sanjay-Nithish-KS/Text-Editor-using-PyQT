
# Text-Editor-using-PyQT

## Features

1. Open         
2. Save        
3. Save As     
4. Undo
5. Redo
6. Edit (Cut, Copy, Paste)
7. Find
8. Find and Replace

## Requirements

1. Python 3.6
			https://www.python.org/downloads/release/python-360/
3. PyQT5 
			`pip install PyQt5`
